连接你的 B站 账号

让 OpenBiliClaw 读到你的观看历史,推荐才懂你。最省事的方式是装浏览器扩展,它会自动同步你的 B站 登录,无需复制 cookie。

  1. 安装 OpenBiliClaw 浏览器扩展（Chrome / Edge）
  2. 在浏览器里登录 bilibili.com
  3. 扩展会自动把登录同步到本地后端
正在等待 B站 登录同步…（装好扩展并登录后会自动变绿）
没装扩展也行——可以先跳过,之后在主界面随时连接。
